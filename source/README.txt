--- HƯỚNG DẪN CHẠY CODE ---

1. Bật Xampp và Start Apache + MySQL.

2. Mở trình duyệt web và truy cập http://localhost/phpmyadmin , chọn nhập file cơ sở dữ liệu tên "data.sql" trong bài nộp.
(File đã bao gồm câu lệnh create database, GV không cần tạo database trước)

3. Tại thư mục chứa code, mở Command Prompt và chạy câu lệnh 'php artisan serve'.

4. Ctrl + click vào link để mở trang web. Khi mở trang web mặc định sẽ là trang mua hàng của khách hàng.

5. Để vào trang quản lý, ta thêm '/admin' vào phía sau đường link và đăng nhập với tên đăng nhập "admin" và mật khẩu "123456".

Lưu ý: cuối video demo có hướng dẫn cụ thể hơn.
